### 补充：为何大量赛季后仍会出现「对局数为 0」的形状？

- 本次状态中锚定条目总数: **240**；其中对局数为 0 的条目: **115**。
- 关键原因是运行机制：**每赛季每个物品只选择 1 名锚定代表**进入当季活跃玩家集合参与匹配与爬山（实现位于 `AnchoredRepresentativeScheduler.SelectRepresentatives`）。未被选中的形状，本赛季不会产生任何对局增长。
- 在“非探索”分支下代表选择使用 **softmax(ELO)** 加权抽样；当某个形状的 ELO 明显高于其它形状时，softmax 会非常尖锐，导致低分形状几乎选不到，从而长期停留在 **ELO≈1500、对局数=0** 的冷启动状态。
- 另一个常见原因是：某些锚定形状在较晚赛季才被创建/重启出来，而在保存状态后程序停止，导致它们还未来得及被抽到代表。
